# Intern report shape

Keep it concise and scannable. The `---ANALYSIS_START---` markdown block should contain:

1. **Metrics table** — a compact markdown table of the key figures (price, change, any
   indicators provided).
2. **Technicals** — 2-4 sentences on trend and momentum from the data.
3. **Macro** — 1-2 sentences tying in any macro/news context.
4. **Short-term outlook** — 2-3 sentences on the next ~30 days. No invented price targets;
   frame qualitatively or with ranges grounded in the data.

Aim for roughly 250-450 words of analysis. Cite specifics from the inputs.
