# King report shape

Comprehensive and structured. The `---ANALYSIS_START---` markdown block should contain:

1. **Metrics table** — key figures (price, change, indicators).
2. **Highlights** — bulleted standout findings.
3. **Technicals** — trend, momentum, key levels (support/resistance) from the data.
4. **On-chain / flows** — any flow, volume, or positioning context from news_sentiment.
5. **Macro** — relevant macro backdrop.
6. **News** — the most material recent developments, cited.
7. **Scenarios** — explicit Bull / Base / Bear cases, each with a probability or
   percentage move framing grounded in the data.
8. **Key levels** — the price levels that matter.
9. **Summary** — a tight closing read.

Aim for roughly 700-1200 words of analysis. Every claim should trace to the provided
inputs and sources.
